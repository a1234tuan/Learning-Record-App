# 408 备考日志 App —— 完整设计构想

---

## 一、这是个什么样的 App

这是一款**纯属个人使用**的 408 考研备考日志工具，主要在**手机（安卓）和 Windows 电脑浏览器**上使用。它的核心定位是：**用最顺手的方式，把每天学习的过程、思考、错题、感悟统统记下来，方便日后回顾、复习和总结。**

它不需要联网，不依赖任何服务器，所有内容都保存在本地。没有账号系统，没有分享功能，没有社交属性——它就是属于"我自己"的一个安静的备考伴侣。

它的灵感来源于 Notion 的优雅结构和 Obsidian 的本地化理念，但又比它们更聚焦——它只为"408 备考"这一件事服务。

---

## 二、整体感觉是什么样

打开这个 App，会有一种"清爽、专注、有归属感"的感觉。

界面以简洁为主，配色温和，不刺眼。白天用浅色背景，晚上用深色背景护眼。字体清晰好读，间距舒服。没有花哨的动画，没有弹窗广告，没有任何让人分心的元素。

它的核心是**让记录变得毫无负担**——想记什么就记什么，怎么方便怎么来。无论是匆忙记一句感悟、贴一张错题截图、写一段心情、还是认真整理一个知识点，它都能从容承接。

打开 App 的第一眼，看到的应该就是"今天"——今天的日期、今天已经记录的内容、今天剩下要做的事。让用户立刻进入"现在"，而不是被复杂的菜单分散注意力。

## 三、记录是核心：能记什么、怎么记

### 1. 一切以"日"为基本单位
每一天都是一个独立的容器。点击某一天，就能看到那天记的所有内容。这种"按天组织"的方式很自然，符合人对时间的直觉，方便日后回顾"我那天在干嘛"。

### 2. 一天里可以记很多种内容
一天的日志不是一篇死板的文章，而是可以由很多"块"组成的。这些"块"可以是：

- **一段文字**：可以是学习总结、思考、心情、感悟
- **一段 Markdown 富文本**：支持标题、列表、加粗、引用、分割线等
- **一张图片**：可以是错题截图、笔记拍照、知识点截图、思维导图等
- **一个附件**：比如 PDF 真题、Word 笔记、其他参考文档
- **一段代码**：408 涉及数据结构和操作系统，写代码需要语法高亮
- **一个数学公式**：408 数学公式特别多，输入 LaTeX 后能自动渲染成漂亮的公式
- **一道错题**：这是特殊的一种块，下面单独说
- **一个学习时长记录**：比如"今天学了数学 3 小时"，可以自动累计统计
- **一个待办清单**：比如"明天要做的 5 件事"，可以打勾完成
- **一段引用 / 名言**：背书时遇到的好句子、考研路上的精神力量

这些块可以**自由拖动顺序、随意组合**，就像 Notion 那样灵活。

### 3. 写起来要顺手
写作的体验必须流畅：

- **手机端**：能直接拍照插入图片、能用语音输入文字、能复制粘贴图片、键盘能切换 Markdown 快捷符号
- **电脑端**：粘贴板里有截图能直接 Ctrl+V 贴进来、有快捷键能快速插入各种块、能拖拽附件文件进来
- **公式输入**：输入两个美元符号包裹的 LaTeX，自动变成数学公式
- **自动保存**：写到一半切出去不会丢，自动保存到本地

### 4. 模板让记录更省事
为了避免每天打开都不知道写什么，提供几种**预设模板**，一键创建：

- **每日学习日志模板**：包含"今日学了什么 / 错题记录 / 卡点疑问 / 明日计划"等区块
- **错题记录模板**：题目截图 / 错误答案 / 正确解法 / 错误原因 / 涉及知识点 / 下次复习时间
- **知识点整理模板**：核心定义 / 关键公式 / 例题 / 易错点
- **每周复盘模板**：本周学了什么 / 完成情况 / 遇到的问题 / 下周计划
- **每月复盘模板**：本月整体进度 / 模考成绩 / 心态总结 / 策略调整

模板只是"快速起点"，可以随时改、随时加内容，不是死板的格式。

## 四、错题本是灵魂功能

错题本不是"普通日志里随便记几道错题"那么简单。它是一个**独立的、贯穿整个备考期的**功能模块。

### 1. 每道错题是一张独立的"卡片"
每一道错题，相当于一张小卡片，上面记着：

- 题目（可以是文字 + 图片）
- 这道题来自哪里（比如"2018 真题第 22 题"）
- 我当时怎么做的、错在哪
- 正确解法是什么
- 涉及的知识点（可以打标签关联）
- 我的反思和总结
- 这道题的难度等级（自己标）
- 当前掌握程度（待复习 / 复习中 / 已掌握）

### 2. 自动安排复习时间（艾宾浩斯曲线）
记录一道错题后，App 会**自动安排几个复习时间点**：1 天后、3 天后、7 天后、15 天后、30 天后。

每次到了复习时间，错题本里会显示"今天需要复习的题"。重做一遍，如果还会错，就回到第一阶段重新复习；如果会做了，就推进到下一阶段。**完全自动化的间隔重复系统**，让错题真的不会"白记"。

### 3. 可以按维度筛选错题
比如：
- 按科目（数学 / 政治 / 英语 / 408）
- 按章节（高数 / 线代 / 数据结构 / 操作系统……）
- 按知识点标签（中值定理 / 进程同步 / 树……）
- 按掌握程度（只看还没掌握的）
- 按时间（最近一周记录的）

随时一键调出"我想看的错题"。

### 4. 错题统计
能看到：
- 我一共记了多少道错题
- 哪个章节错得最多（暴露知识薄弱区）
- 我的"已掌握"比例（看到自己的进步）

## 五、组织和检索：让以前记的内容随时找得到

一个 3 个月以上的备考期，会积累**几百条日志、几百道错题、上千张图片**。如果找不到以前的东西，记了等于没记。所以组织和检索必须很强。

### 1. 标签系统是骨架
每一篇日志、每一道错题，都可以打多个标签。标签可以分层级：

- 一级标签：科目（`#数学` `#英语` `#政治` `#408`）
- 二级标签：章节（`#高数` `#线代` `#数据结构` `#操作系统`）
- 三级标签：知识点（`#中值定理` `#动态规划` `#页面置换`）
- 状态标签（`#错题` `#待复习` `#已掌握` `#重点` `#疑问`）
- 心情标签（`#状态好` `#焦虑` `#突破` `#瓶颈`）

标签之间可以多选筛选，比如同时筛选 `#数学` + `#错题` + `#待复习`。

### 2. 全文搜索
能搜任何字。输入"中值定理"，所有提到这四个字的日志、错题、知识点都会出来。**支持中文，支持模糊匹配**。

### 3. 多种查看方式
不是只有"列表"一种看法：

- **日历视图**：以日历形式展示哪天有记录，可以看到自己的"打卡"轨迹（像 GitHub 那种绿色方块热力图）
- **列表视图**：按时间倒序，最新的在最上面
- **时间轴视图**：以时间轴形式展示自己整个备考的关键节点
- **标签视图**：按某个标签筛选出所有相关内容
- **错题本视图**：单独的错题管理界面

### 4. 收藏与置顶
特别重要的内容（比如自己写得很好的总结、特别精辟的反思），可以**收藏或置顶**，永远在首页能快速找到。

### 5. 内容之间可以互相引用（双向链接）
比如某天的日志里提到了"中值定理"，可以把这四个字变成一个链接，点击它就跳转到"中值定理"这个知识点页面。这个知识点页面又能反向看到"有哪些日志提到了我"。

这样形成一张**知识网络**，比单纯的文件夹结构强大得多。**就是 Notion 和 Obsidian 的那种感觉。**

## 六、统计与可视化：看到自己的进步

记录了这么多，要能"看到"自己的努力，才有持续的动力。

### 1. 打卡热力图
首页或统计页有一张类似 GitHub 贡献图的图——一年 365 天，每天一个方块，颜色深浅代表当天的学习量。看着自己一格一格变绿，是巨大的成就感。

### 2. 学习时长统计
每天可以记录"我今天学了多少小时数学、多少小时政治"。App 自动汇总：

- 今日学习时长
- 本周 / 本月学习时长
- 各科目学习时长占比（饼图）
- 学习时长趋势（折线图）

### 3. 错题本统计
- 累计错题数
- 已掌握比例
- 各章节错题分布（暴露弱点）
- 复习完成率

### 4. 备考进度可视化
可以**预先录入 408 各章节列表**，然后给每个章节打"已学/复习中/已掌握"的标记，看到自己整体进度条。一眼看清"还有哪些没学"。

### 5. 关键里程碑
能记录一些大事件，比如：
- "完成数学第一轮复习"
- "模考第一次上 100"
- "背完 5000 个单词"

这些里程碑会在时间轴上特别标注，回头看会很感动。

## 七、本地存储与数据安全

这是这个 App 最核心的设计原则之一。

### 1. 所有数据都在本地
没有云端，没有账号。所有日志、图片、附件都保存在手机或电脑的浏览器本地存储里。**没有任何隐私泄露的可能。**

### 2. 但要防止数据丢失
本地存储最大的风险是丢失（清浏览器缓存、换手机、误操作）。所以必须有强大的备份机制：

- **一键导出全量数据**：导出为一个压缩包（包含所有日志的 Markdown 文件 + 图片 + 附件 + 错题数据库），可以保存到云盘、U 盘、邮箱
- **一键导入恢复**：换设备或重装时，把备份文件导入就能完全恢复
- **定期备份提醒**：比如每周提醒一次"记得备份哦"
- **导出单篇 / 单章节**：也可以单独导出某天的日志、某个知识点

### 3. 多端数据同步（可选但重要）
因为既有手机用，又有电脑用，所以希望数据能在两边同步。但又不想用服务器。**解决方案是用第三方网盘做中转**：

- 把数据库文件存到坚果云 / OneDrive / 阿里云盘的文件夹里
- 手机和电脑都从这个文件夹读写
- 修改后自动同步到云盘
- 实现"无服务器但能同步"

或者更简单的方式：**手动导出 + 导入**。隔几天把手机的数据导出，发到电脑导入一次。

## 八、手机端和电脑端的差异化设计

虽然是同一个 App（PWA），但手机和电脑用起来场景不同，要做差异化体验。

### 手机端的重点
手机适合**碎片化、快速记录**：

- **首页就是写**：打开就能直接开始记，不用层层点击
- **快速插入按钮**：底部一排大图标——拍照、图片、文字、录音、错题
- **语音输入**：长按麦克风说话转文字
- **简洁的卡片式浏览**：左右滑动看历史日志
- **离线可用**：地铁里没网也能用
- **可以"安装"到桌面**：从浏览器添加到主屏幕，看起来就像原生 App
- **大字体、大按钮**：单手操作友好

### 电脑端的重点
电脑适合**深度整理、系统复盘**：

- **三栏布局**：左侧导航、中间编辑、右侧大纲或相关内容
- **键盘快捷键**：所有操作都能用快捷键完成
- **大屏多任务**：可以一边看 PDF 真题，一边在 App 里记录
- **批量操作**：可以批量打标签、批量导出、批量整理
- **丰富的视图切换**：日历、列表、看板、时间轴随意切换

## 九、其他贴心小功能

### 1. 番茄钟集成
学习时可以启动一个番茄钟（25 分钟专注 + 5 分钟休息）。一轮结束时，App 弹出"刚才这 25 分钟你做了什么？"，引导你记录。把学习节奏和记录天然结合。

### 2. 每日提醒
设置一个时间（比如晚上 10 点），App 提醒"今天还没记日志哦"，避免漏记。

### 3. 复习提醒
错题到了复习时间，给个提醒。也可以是每周一次的"本周复盘提醒"。

### 4. 引言与鼓励
打开 App 时，偶尔显示一句鼓励的话或备考名言。不是鸡汤式的，而是简短、克制、有力的。

### 5. 暗黑模式
晚上自习室或卧室学习时，自动切换深色模式。

### 6. 字体与排版自定义
可以选喜欢的字体（衬线 / 无衬线 / 等宽）、调整字号、调整行间距。让自己的工具有自己的"味道"。

### 7. 锁屏密码（可选）
如果手机不想让别人看到自己的备考日志，可以设置一个 4 位数密码或指纹。

### 8. 倒计时
首页显示"距离 408 考试还有 XX 天"。每天看一眼，提醒自己时间的宝贵。

## 十、它不应该是什么样

最后说一下**它绝对不能变成什么样**——这同样重要：

- **不能变成社交工具**：没有分享、没有排行榜、没有点赞、没有评论。这是属于一个人的安静空间
- **不能变成花哨的玩具**：没有过度动画、没有炫技特效、没有打开就弹的广告或推荐
- **不能变成"做工具"的借口**：功能要克制，宁可少做几个功能，也不要做一个让人忙着研究界面而不是学习的工具
- **不能让人有"必须每天打开"的压力**：偶尔忘了记一天也没事，工具不应该惩罚使用者
- **不能依赖网络**：地铁里、自习室没网时，也要能完全正常使用
- **不能让数据成为"人质"**：随时能完整导出自己的数据，离开这个工具也不会丢失任何东西

---

## 总结一句话

> 这是一个**纯本地、跨端、简洁、安静**的备考伴侣，用 Notion 式的灵活结构承接备考过程中的所有记录，用错题本和间隔重复系统帮助高效复习，用统计和可视化让努力被看见，最终成为一份**属于自己的、值得永久珍藏的考研回忆录**。

它的终极目标不只是"帮我考上"，而是让我考完研后回头翻一翻，能清楚地看到：**那段日子，我每一天都在做什么，都在想什么，怎么一步步走过来的**。
